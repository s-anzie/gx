Getting Started

Quickstart

From zero to a running API in under a minute.

1

Create a new module

Run these commands in your project directory.

terminal shell 
# Create module
mkdir myapp && cd myapp
go mod init myapp

# Install GX
go get github.com/s-anzie/gx@latest
2

Write main.go

A minimal app with one route.

main.go go 
package main

import (
    "github.com/s-anzie/gx"
    "github.com/s-anzie/gx/core"
)

func main() {
    app := gx.New()

    app.GET("/ping", func(c *core.Context) core.Response {
        return c.JSON(map[string]any{"message": "pong"})
    })

    // Listen handles SIGTERM/SIGINT automatically
    app.Listen(":8080")
}
3

Run it

GX logs startup info and handles SIGTERM / SIGINT automatically.

terminal shell 
# Start the server
go run main.go

# In another terminal
curl http://localhost:8080/ping
# → {"message":"pong"}
✅

Listen listens for SIGTERM and SIGINT and performs a graceful shutdown (default 10 s timeout)

With boot hooks

Call app.Boot(ctx) before listening to run lifecycle hooks (database connections, plugin initialisation, etc.):

main.go go 
func main() {
    app := gx.New()

    app.OnBoot(func(ctx context.Context) error {
        return db.Connect(ctx)
    })

    app.GET("/ping", pingHandler)

    // Run boot hooks first, then start serving
    if err := app.Boot(context.Background()); err != nil {
        log.Fatal(err)
    }
    app.Listen(":8080")
}
