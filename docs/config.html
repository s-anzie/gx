Features

Config from Env

gx.LoadConfig[T]() is a zero-dependency, type-safe environment variable loader. Annotate a plain struct with env, default, and required tags — no external library needed.

Define your config struct

config.gogo 
type AppConfig struct {
    DatabaseURL string        `env:"DATABASE_URL" required:"true"`
    RedisURL    string        `env:"REDIS_URL"    default:"redis://localhost:6379"`
    Port        int           `env:"PORT"         default:"8080"`
    Debug       bool          `env:"DEBUG"        default:"false"`
    Timeout     time.Duration `env:"TIMEOUT"      default:"30s"`
    Workers     int           `env:"WORKERS"      default:"4"`

    // Nested structs are supported
    Auth struct {
        Secret     string `env:"AUTH_SECRET"      required:"true"`
        Expiry     int    `env:"AUTH_EXPIRY_SECS" default:"3600"`
    }
}

Loading config

main.gogo 
cfg, err := gx.LoadConfig[AppConfig]()
if err != nil {
    log.Fatalf("config: %v", err)
}

app := gx.New()
// … register routes, pass cfg to dependencies …
app.ListenWithGracefulShutdown(fmt.Sprintf(":%d", cfg.Port))

Struct tag reference

TagDescription
env:"VAR_NAME"Environment variable name to read
default:"value"Value used when the env var is absent
required:"true"Returns an error if the env var is absent and no default is set

Supported field types

LoadConfig performs automatic type conversion from the string environment variable to native Go types:

TypeExample value
string"postgres://localhost/db"
int, int64"8080"
float64"3.14"
bool"true", "1", "false", "0"
time.Duration"30s", "5m", "1h"
