Plugins

The plugin system is GX's extension mechanism. Plugins hook into the application at boot time, per-request, on errors, and on route registration.

Plugin Interfaces

All plugins implement the base Plugin interface. Additional capability is added by implementing one or more optional sub-interfaces.

go
// Base interface — every plugin must implement this
type Plugin interface {
    Name() string
    OnBoot(app *gx.App) error
    OnShutdown(ctx context.Context) error
}

// RequestPlugin — intercepts every HTTP request
type RequestPlugin interface {
    Plugin
    OnRequest(c *gx.Context, next core.Handler) core.Response
}

// ErrorPlugin — intercepts unhandled errors
type ErrorPlugin interface {
    Plugin
    OnError(c *gx.Context, err gx.AppError) core.Response
}

// RoutePlugin — called once per registered route at boot
type RoutePlugin interface {
    Plugin
    OnRoute(route gx.RouteInfo)
}
InterfaceUse case
PluginBase; lifecycle only (boot/shutdown)
RequestPluginAuth, CORS, rate limiting, caching, compression
ErrorPluginCustom error formatting, Sentry/error tracking
RoutePluginOpenAPI spec generation, route auditing

Installing a Plugin

Use app.Install() to add any plugin. For RequestPlugins you can also use app.Use() (which installs with a default priority).

go
import (
    "github.com/s-anzie/gx"
    "github.com/s-anzie/gx/plugins/cors"
    "github.com/s-anzie/gx/plugins/auth"
)

app := gx.New()

// Install with default priority
app.Install(cors.New(cors.DefaultConfig()))

// Install with explicit priority (higher = runs first)
app.Install(auth.JWT(auth.Secret("mysecret")), gx.PluginPriority(750))

Writing a Custom Plugin

Implement the interface(s) you need and pass the plugin to app.Install().

go
package main

import (
    "context"
    "log"

    "github.com/s-anzie/gx"
    "github.com/s-anzie/gx/core"
)

type TimingPlugin struct{}

func (p *TimingPlugin) Name() string { return "timing" }

func (p *TimingPlugin) OnBoot(app *gx.App) error {
    log.Println("timing plugin booted")
    return nil
}

func (p *TimingPlugin) OnShutdown(ctx context.Context) error {
    return nil
}

// OnRequest makes this a RequestPlugin — runs on every request
func (p *TimingPlugin) OnRequest(c *gx.Context, next core.Handler) core.Response {
    start := time.Now()
    resp := next(c.Context)
    c.Log().Info("request timing", "duration", time.Since(start))
    return resp
}

func main() {
    app := gx.New()
    app.Install(&TimingPlugin{})
    app.ListenWithGracefulShutdown(":8080")
}

Error Plugin

Implement OnError to customize how unhandled errors are rendered — useful for integrating with error tracking services.

go
type SentryPlugin struct{}

func (p *SentryPlugin) Name() string { return "sentry" }
func (p *SentryPlugin) OnBoot(app *gx.App) error   { return nil }
func (p *SentryPlugin) OnShutdown(ctx context.Context) error { return nil }

func (p *SentryPlugin) OnError(c *gx.Context, err gx.AppError) core.Response {
    // Report to Sentry
    sentry.CaptureException(err)

    // Return the default error response
    return err.ToResponse()
}

app.Install(&SentryPlugin{})

Route Plugin

OnRoute is called once per route at boot. Use it to build route inventories, enforce naming conventions, or generate documentation.

go
type AuditPlugin struct{}

func (p *AuditPlugin) Name() string { return "audit" }
func (p *AuditPlugin) OnBoot(app *gx.App) error   { return nil }
func (p *AuditPlugin) OnShutdown(ctx context.Context) error { return nil }

func (p *AuditPlugin) OnRoute(route gx.RouteInfo) {
    log.Printf("registered route: %s %s", route.Method, route.Path)
}

app.Install(&AuditPlugin{})
