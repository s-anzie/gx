Plugins

Rate Limiting

The ratelimit plugin enforces request-rate limits using a sliding-window counter. It adds standard X-RateLimit-* headers and returns 429 Too Many Requests when the limit is exceeded.

terminal
go get github.com/s-anzie/gx/plugins/ratelimit

Basic Usage

go
import (
    "time"
    "github.com/s-anzie/gx"
    "github.com/s-anzie/gx/plugins/ratelimit"
)

app := gx.New()

// 100 requests per minute per IP address
app.Install(ratelimit.New(
    ratelimit.PerIP(100, time.Minute),
))

Options

OptionDescription
ratelimit.PerIP(limit int, window time.Duration)Rate limit by client IP address. Sets Limit, Window, and a default IP-based KeyFunc.
ratelimit.PerUser(limit int, window time.Duration)Rate limit by authenticated user ID (falls back to IP)
ratelimit.OnExceeded(fn func(*gx.Context) core.Response)Custom response when limit is exceeded
ratelimit.WithBackend(b Backend)Custom storage backend

Rate Limit Headers

These headers are set on every response (not just when the limit is exceeded).

HeaderDescription
X-RateLimit-LimitMaximum requests allowed in the window
X-RateLimit-RemainingRemaining requests in the current window
Retry-AfterSeconds until the window resets (only on 429 responses)

Custom Exceeded Response

go
app.Install(ratelimit.New(
    ratelimit.PerIP(60, time.Minute),
    ratelimit.OnExceeded(func(c *gx.Context) core.Response {
        return c.Fail(gx.E(429, "RATE_LIMITED",
            "Too many requests. Please slow down."))
    }),
))

Per-User Rate Limiting

PerUser extracts the user identity from the typed context (set by the auth plugin). It falls back to IP address if no user identity is found.

go
// Install auth first, then rate limit per user
app.Install(auth.JWT(auth.Secret("secret")), gx.PluginPriority(750))
app.Install(ratelimit.New(
    ratelimit.PerUser(1000, time.Hour),
), gx.PluginPriority(800))

Custom Backend

Implement ratelimit.Backend to use Redis or another distributed store for rate limiting across multiple instances.

go
// ratelimit.Backend interface
type Backend interface {
    Increment(key string, window time.Duration, limit int) (count int, exceeded bool, err error)
    Reset(key string) error
}

app.Install(ratelimit.New(
    ratelimit.PerIP(100, time.Minute),
    ratelimit.WithBackend(&RedisRateLimitBackend{client: redisClient}),
))
