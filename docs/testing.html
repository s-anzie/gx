Testing

The gxtest package lets you test GX handlers, routers, and full apps without starting a real HTTP server. It wraps net/http/httptest with a fluent API and rich assertion helpers.

go
import "github.com/s-anzie/gx/gxtest"

Supported Targets

gxtest accepts any of these as the first argument to request builders:

TypeDescription
*gx.App / http.HandlerFull application with all middleware and plugins
*core.RouterA router group
core.HandlerA single handler function

Building Requests

Each method builder returns a *RequestBuilder with a fluent configuration API.

go
// Available builders
gxtest.GET(target, path)
gxtest.POST(target, path)
gxtest.PUT(target, path)
gxtest.PATCH(target, path)
gxtest.DELETE(target, path)

// Fluent configuration
builder.
    Header("Authorization", "Bearer token"). // Add request header
    Cookie("session", "abc123").             // Add cookie
    Query("page", "2").                      // Add query param
    JSON(body).                              // Set JSON body + Content-Type
    XML(body).                               // Set XML body + Content-Type
    Body(reader, "text/plain").              // Set raw body + Content-Type
    BodyString("raw text", "text/plain").    // Set string body
    .Do(t)                                    // Execute — returns *RecordedResponse

Assertions

*RecordedResponse provides accessor methods and assertion helpers that call t.Error on failure.

MethodDescription
.Status() intReturns the HTTP status code
.Body() stringReturns the response body as a string
.Header(key) stringReturns a response header value
.AssertStatus(t, n)Asserts status code == n
.AssertHeader(t, key, val)Asserts header equals value
.AssertHeaderContains(t, key, sub)Asserts header contains substring
.AssertBodyContains(t, sub)Asserts body contains substring
.AssertBodyEquals(t, s)Asserts body equals string exactly
.AssertJSON(t, &out, fn)Unmarshals body into out, then runs fn for assertions
.AssertError(t, code)Asserts GX structured error with the given code
.AssertErrorStatus(t, code, status)Asserts both error code and status code
.AssertCookie(t, name)Asserts a cookie with the given name is set
.AssertNoCookie(t, name)Asserts no cookie with the given name is set
.AssertContentType(t, ct)Asserts Content-Type contains ct

Examples

Testing a handler directly

go
func TestGetUser(t *testing.T) {
    handler := func(c *core.Context) core.Response {
        return c.JSON(map[string]string{"id": c.Param("id")})
    }

    res := gxtest.GET(handler, "/users/42").Do(t)

    res.AssertStatus(t, 200)
    res.AssertContentType(t, "application/json")
    res.AssertBodyContains(t, "42")
}

Testing a full app

go
func TestCreateUser(t *testing.T) {
    app := gx.New()
    app.POST("/users", func(c *core.Context) core.Response {
        var body struct {
            Name string `json:"name"`
        }
        if resp := gx.ValidateOrFail(c, &body); resp != nil {
            return resp
        }
        return core.WithStatus(c.JSON(body), 201)
    })

    res := gxtest.POST(app, "/users").
        JSON(map[string]string{"name": "Alice"}).
        Do(t)

    res.AssertStatus(t, 201)

    var got struct{ Name string }
    res.AssertJSON(t, &got, func() {
        if got.Name != "Alice" {
            t.Errorf("expected Alice, got %s", got.Name)
        }
    })
}

Testing an error response

go
func TestNotFound(t *testing.T) {
    app := gx.New()
    app.GET("/users/:id", func(c *core.Context) core.Response {
        return c.Fail(gx.ErrNotFound)
    })

    res := gxtest.GET(app, "/users/999").Do(t)

    res.AssertErrorStatus(t, "NOT_FOUND", 404)
}

Testing with auth headers

go
func TestProtectedRoute(t *testing.T) {
    token, _ := auth.GenerateToken("secret", jwt.MapClaims{
        "sub": "user-1",
        "exp": time.Now().Add(time.Hour).Unix(),
    })

    res := gxtest.GET(app, "/api/me").
        Header("Authorization", "Bearer "+token).
        Do(t)

    res.AssertStatus(t, 200)
}
