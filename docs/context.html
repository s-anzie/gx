Context & Request

*core.Context is your handle on the entire request lifecycle. It exposes request data, response builders, and per-request storage.

Reading the request

MethodReturnsNotes
c.Param("id")stringNamed URL parameter
c.Query("q")stringQuery string value
c.QueryDefault("q","val")stringQuery with fallback
c.Header("Authorization")stringRequest header

Per-request store

Use c.Set / c.Get to pass data between middleware and handlers:

store.gogo 
// In middleware
c.Set("userID", claims.Subject)

// In handler
uid, _ := c.Get("userID")
// or panic on missing:
uid := c.MustGet("userID")

Typed store

Use the generic helpers for type-safe access:

typed_store.gogo 
// Set a typed value (no interface{})
gx.SetTyped[UserClaims](c, claims)

// Get — returns *T (nil if missing or wrong type)
claims := gx.Typed[UserClaims](c)

// TryTyped — returns (*T, bool)
claims, ok := gx.TryTyped[UserClaims](c)
