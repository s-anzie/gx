OpenAPI

The OpenAPI plugin generates an OpenAPI 3.1 specification from route contracts and serves an interactive docs UI. It supports Scalar (default), Swagger UI, and Redoc.

terminal
go get github.com/s-anzie/gx/plugins/openapi

Basic Setup

go
import (
    "github.com/s-anzie/gx"
    "github.com/s-anzie/gx/plugins/openapi"
)

app := gx.New()

app.Install(openapi.New(
    openapi.Title("My API"),
    openapi.Version("1.0.0"),
    openapi.Description("REST API powered by GX"),
))

// Spec available at:  GET /openapi.json
// Docs UI available at: GET /docs

Documenting Routes with Contracts

The spec is built from contracts attached to routes. A contract describes the summary, tags, input schemas, output schema, and possible errors for a route.

go
type CreateUserBody struct {
    Name  string `json:"name"  validate:"required"`
    Email string `json:"email" validate:"required,email"`
}

type UserResponse struct {
    ID    string `json:"id"`
    Name  string `json:"name"`
    Email string `json:"email"`
}

var createUserContract = gx.Contract{
    Summary:     "Create a user",
    Description: "Creates a new user account.",
    Tags:        []string{"Users"},
    Body:        gx.Schema[CreateUserBody](),
    Output:      gx.Schema[UserResponse](),
    Errors:      []gx.AppError{gx.ErrValidation, gx.ErrConflict},
}

app.POST("/users", createUserContract, func(c *core.Context) core.Response {
    var body CreateUserBody
    if resp := gx.ValidateOrFail(c, &body); resp != nil {
        return resp
    }
    // ... create user
    return c.JSON(UserResponse{ID: "1", Name: body.Name, Email: body.Email})
})

Choosing a UI

go
// Scalar (default) — modern, clean
app.Install(openapi.New(openapi.UI(openapi.Scalar)))

// Swagger UI — familiar
app.Install(openapi.New(openapi.UI(openapi.Swagger)))

// Redoc — documentation-focused
app.Install(openapi.New(openapi.UI(openapi.Redoc)))

Multiple Servers

go
app.Install(openapi.New(
    openapi.Title("My API"),
    openapi.Version("2.0.0"),
    openapi.AddServer("https://api.example.com", "Production"),
    openapi.AddServer("https://staging-api.example.com", "Staging"),
    openapi.AddServer("http://localhost:8080", "Development"),
))

Disabling in Production

go
app.Install(openapi.New(
    openapi.Title("My API"),
    openapi.DisableInProduction(app), // disabled when GX_ENV=production
))
