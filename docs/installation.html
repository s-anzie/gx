Getting Started

Installation

Install the main module and any plugins you need:

terminal shell 
# Core framework (required)
go get github.com/s-anzie/gx@latest

# Plugins (install only what you use)
go get github.com/s-anzie/gx/plugins/cors
go get github.com/s-anzie/gx/plugins/auth
go get github.com/s-anzie/gx/plugins/compress
go get github.com/s-anzie/gx/plugins/cache
go get github.com/s-anzie/gx/plugins/ratelimit
go get github.com/s-anzie/gx/plugins/openapi

App options

gx.New() accepts functional options to configure the application at startup.

OptionDefaultDescription
WithEnvironment(env)DevelopmentSets app environment: Development, Staging, Production
ReadTimeout(d)15sMax duration for reading an entire request
WriteTimeout(d)15sMax duration before timing out response writes
ShutdownTimeout(d)10sMax wait time for graceful shutdown
MaxBodySize(n)10 MBRequest body size limit in bytes
TrustedProxies(...)—IP/CIDR list of trusted reverse proxies
WithDevTLS(hosts...)—Self-signed TLS for local development
IgnoreEnv()—Disable GX_* environment variable overrides
main.go go 
app := gx.New(
    gx.WithEnvironment(gx.Production),
    gx.ReadTimeout(30*time.Second),
    gx.WriteTimeout(30*time.Second),
    gx.MaxBodySize(32 << 20), // 32 MB
    gx.TrustedProxies("10.0.0.0/8"),
)

Environment variables

GX reads GX_* env vars at startup, after options are applied. Use IgnoreEnv() to disable this.

VariableExampleDescription
GX_ENVproductionSets the runtime environment
GX_MAX_BODY_SIZE32MBMax request body size (supports B/KB/MB/GB)
GX_TRUSTED_PROXIES10.0.0.1,10.0.0.2Comma-separated trusted proxy IPs
