Features

Early Hints

RFC 8297 HTTP 103 Early Hints lets the server tell the browser what sub-resources to fetch before the final response is ready. This overlaps DNS/TCP/TLS with server processing, reducing page load time.

Early Hints require HTTP/2 or HTTP/3. On HTTP/1.1 connections c.EarlyHints() is a no-op.

Basic usage

handler.gogo 
func GetPage(c *core.Context) core.Response {
    // Send 103 hints immediately — browser starts loading resources
    c.EarlyHints(
        core.Preload("/assets/app.css",  "style"),
        core.Preload("/assets/app.js",   "script"),
        core.Preconnect("https://api.example.com"),
    )

    // … slow database query, template rendering …
    page := renderPage(c)

    return c.HTML(page)
}

Link constructors

Functionrel=Description
core.Preload(uri, as)preloadLoad a critical sub-resource (style, script, image, font, fetch)
core.Prefetch(uri)prefetchHint for a resource needed on a likely next navigation
core.Preconnect(uri)preconnectOpen TCP + TLS connection to origin early
core.DNSPrefetch(uri)dns-prefetchResolve the DNS for an origin in advance

Cross-origin resources

Set CrossOrigin: true on a core.Link directly for cross-origin hints (e.g., fonts hosted on a CDN):

cross_origin.gogo 
c.EarlyHints(core.Link{
    URI:         "https://cdn.example.com/font.woff2",
    Rel:         "preload",
    As:          "font",
    CrossOrigin: true,
})
