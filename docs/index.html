main.go 
package main

import (
    "github.com/s-anzie/gx"
    "github.com/s-anzie/gx/plugins/cors"
)

func main() {
    app := gx.New()

    app.Install(cors.New())

    app.GET("/users/:id", func(c *gx.Context) core.Response {
        id := c.Param("id")
        return c.JSON(gx.Map{"id": id, "ok": true})
    })

    app.Boot(":8080")
}

Core Features

Everything you need.
Nothing you don't.

GX gives you a thin, fast core and a rich set of composable features — each opt-in.

⚡

Radix-tree Router

Sub-microsecond route resolution with named params, wildcards, and method-aware routing.

🧩

Composable Middleware

Chain middleware globally, per-group, or per-route. Full control over execution order.

🔌

Priority Plugin System

Install plugins at any time. Priority-ordered execution ensures consistent behavior.

🛡️

Type-safe Contracts

Declare request/response schemas with Contract for automatic OpenAPI generation.

🌐

HTTP/2 & HTTP/3

TLS auto-config, QUIC transport (http3 via quic-go), and HTTP/3 Alt-Svc advertisement.

🔍

Content Negotiation

Automatic JSON/XML/text selection from Accept headers. Language detection included.

✅

Struct Validation

Built-in validation via go-playground/validator with human-readable error messages.

📁

Static File Serving

Serve directories, single files, or embed.FS. Brotli/gzip auto-detection.

🍪

Cookies & Sessions

Typed cookie API with secure defaults. HttpOnly + SameSite=Lax out of the box.

📤

File Uploads

Magic-byte MIME detection. Save, open, or read upload bytes directly from context.

⚙️

Env Config Loader

Generic LoadConfig[T] reads env vars into structs with defaults and validation.

🧪

Testing Utilities

The gxtest package lets you test handlers without starting a server.

Routing & Groups

Expressive, nested routing

Define routes directly on the app, or group them with shared prefixes and middleware. The Mount API lets you attach entire routers.

Named params :id Wildcards * Groups Mount Any()
routing.go 
// Route groups with middleware
api := app.Group("/api", authMiddleware)

v1 := api.Group("/v1")
v1.GET("/users", listUsers)
v1.POST("/users", createUser)
v1.GET("/users/:id", getUser)

// Standalone router — mountable
r := core.NewRouter()
r.GET("/health", healthHandler)

app.Mount("/admin", r, adminOnly)
Validation

Validate once, use anywhere

Use gx.Validate() for standalone validation, or ValidateOrFail() to automatically return a 422 response on failure.

go-playground/validator Human errors Custom rules 422 auto-response
validate.go 
type CreateUser struct {
    Name  string `validate:"required,min=2"`
    Email string `validate:"required,email"`
    Age   int    `validate:"gte=18"`
}

func createUser(c *gx.Context) core.Response {
    var body CreateUser
    c.BindJSON(&body)

    if res, ok := gx.ValidateOrFail(c, &body); !ok {
        return res // 422 with field errors
    }
    // body is valid ✓
    return c.Created(body)
}
