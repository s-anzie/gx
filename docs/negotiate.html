Features

Content Negotiation

GX reads the client's Accept and Accept-Language headers to automatically choose the best response format or language.

c.Negotiate

c.Negotiate(v) picks the best format from application/json, application/xml, and text/plain based on the client's Accept header. JSON is the default when the header is absent or */*. Returns 406 Not Acceptable if no supported format matches.

negotiate.gogo 
func GetUser(c *core.Context) core.Response {
    user := fetchUser(c.Param("id"))
    return c.Negotiate(user)
    // Accept: application/json   → JSON
    // Accept: application/xml    → XML
    // Accept: text/plain         → plain text
    // Accept: text/html          → 406 Not Acceptable
}

c.Accepts

For custom format selection, c.Accepts(candidates...) respects q= quality weights and wildcard patterns like text/*. Returns the best-matching candidate, or "" if none matches.

accepts.gogo 
switch c.Accepts("application/json", "text/csv", "application/pdf") {
case "text/csv":
    return renderCSV(data)
case "application/pdf":
    return renderPDF(data)
default:
    return c.JSON(data)
}

c.AcceptLanguage

c.AcceptLanguage() returns the primary language tag from the Accept-Language header, normalized to a base code (e.g., "fr" for "fr-FR"). Defaults to "en" if the header is absent. Use c.AcceptLanguages() for the full ordered list.

language.gogo 
lang := c.AcceptLanguage()     // "fr", "en", "de", …
langs := c.AcceptLanguages()   // ["fr", "en"] in preference order

// i18n with structured errors
var ErrNotFound = gx.E(404, "NOT_FOUND", "Not found", gx.Translations{
    "fr": "Ressource introuvable",
    "es": "No encontrado",
})

msg := ErrNotFound.MessageFor(c.AcceptLanguage())
