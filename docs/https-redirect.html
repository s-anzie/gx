HTTPS Redirect

GX provides two approaches for enforcing HTTPS: an integrated redirect listener that runs the HTTP and HTTPS servers together, and a standalone middleware for use behind a TLS-terminating proxy.

app.ListenHTTPRedirect

app.ListenHTTPRedirect(httpAddr, httpsAddr) starts a small HTTP server that permanently redirects every request to the equivalent HTTPS URL, while your main server listens on the HTTPS port. Run them in separate goroutines.

https_server.gogo 
func main() {
    app := gx.New()
    app.WithTLS("cert.pem", "key.pem")

    // HTTP :80 → HTTPS :443 redirect (runs in background)
    app.ListenHTTPRedirect(":80", ":443")

    // Main HTTPS server
    if err := app.ListenWithGracefulShutdown(":443"); err != nil {
        log.Fatal(err)
    }
}

gx.HTTPSRedirect middleware

When terminating TLS at a proxy (Nginx, Caddy, a load balancer), use the gx.HTTPSRedirect() middleware. It reads the X-Forwarded-Proto header and issues a 301 redirect if the request arrived over plain HTTP.

proxy_redirect.gogo 
app := gx.New()
app.Use(gx.HTTPSRedirect())
// all other routes …

Development TLS

For local HTTPS with a self-signed certificate use app.WithTLS after calling gx.NewDevTLS() or pointing to existing certificate files:

dev_tls.gogo 
app := gx.New(
    gx.WithDevTLS(),   // self-signed cert for localhost development
)
app.ListenWithGracefulShutdown(":8443")
